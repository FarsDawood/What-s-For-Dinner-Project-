# What's For Dinner?

**"What's For Dinner"** is a dynamic web application designed to solve the daily dilemma of choosing what to cook. It provides users with random recipe suggestions, complete with detailed ingredients, instructions, nutritional facts, and chef's tips.

* **Random Recipe Generator:** Instantly get a new recipe suggestion with a single click.
* **Smart Logic:** The algorithm ensures the same recipe isn't suggested twice in a row.
* **Dynamic Content:** All content (images, text, stats) is updated dynamically using JavaScript without reloading the page.
* **Tabbed Interface:** Organized view for Ingredients, Instructions, Nutrition, and Chef's Tips.
* **Responsive Design:** Fully responsive layout that works perfectly on desktops, tablets, and mobile phones using **Bootstrap 5**.
* **Nutritional Breakdown:** Detailed view of calories, protein, carbs, and more.
* **Custom Styling:** Modern UI with custom scrollbars, color variables, and interactive hover effects.
